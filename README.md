# Olist BI Analytics: Análise de Performance de E-commerce Brasileiro

Projeto de Business Intelligence desenvolvido para a disciplina Projeto de Software da Faculdade Impacta.

## Objetivo
Analisar dados públicos de e-commerce brasileiro utilizando SQL e Power BI, gerando insights sobre vendas, clientes, categorias, distribuição geográfica e desempenho inicial do negócio.

## Ferramentas utilizadas
- VS Code
- SQL
- Power BI
- GitHub
- Trello

## Dataset
Brazilian E-Commerce Public Dataset by Olist

## Estrutura do projeto

data/ → arquivos de dados  
sql/ → consultas SQL  
powerbi/ → arquivo do dashboard  
docs/ → documentos das entregas  
images/ → imagens e prints do projeto  
icons/ → icones do projeto
